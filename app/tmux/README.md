# Tmux

## Prerequisiti

* Tmux

## Installazione

Copia il contenuto del file `tmux.conf` in `~/.tmux.conf`.

```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
tmux
tmux source-file ~/.tmux.conf
```
